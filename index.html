Who I am

What I do

  • I work in FAIR (Facebook AI Research) on the machine translation project No Language Left Behind.
  • As side projects, I develop NLP tools and applications such as:
    • The first neural machine translation system for the Erzya language (paper, blog post in Russian, code)
    • A framework for creating chat bots in Python Dialogic
    • A package for compressing FastText models compress-fasttext
    • Neural network models for Russian language, such as tiny BERT and Russian T5 in the Huggingface repository
    • Skills for the Alice voice assistant dialogic.digital
  • I teach applied mathematics and data analysis, in particular:
    • A course on Probability and Statistics for Data Scientists in the Tel Aviv school of Data Analysis (Y-DATA).
    • An introduction to Data Science for Products and other managers, a course by MIPT and MathsHub
    • Ad-hoc lectures on correct problem statements for data science, natural language processing, basics of mathematical modelling, and other interesting topics.
